    ---------------
    PasswordFinder is an easy-to-use utility that allows to find the text corresponding to a hash.

    3. Installing and Uninstalling
    ------------------------------
    To install PasswordFinder, run Setup.exe from the distribution.
    To uninstall PasswordFinder, run "Add or Remove Programs" from Control Panel, or run Uninstall from Start\All Programs\PasswordFinder.
